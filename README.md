# Crello (crello)

Trello but good

## Instalar a depedências
```bash
yarn
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
quasar dev
```


### Lint the files
```bash
yarn lint
```


### Format the files
```bash
yarn format
```


### Build the app for production
```bash
quasar build
```
